# Monte Carlo Dropout — оценка неопределённости

Код к презентации про байесовскую неопределённость в классификации.

## Суть
Байессовкий метод вычислительно сложный, для упрощения придумали вариационный подход, в котором необходимо приближать апостериорное распределение другим распределением с теми же параметрами. Также было доказано, что можно аппроксимировать апостериорное распределение, если не отключать дропаут в моменте инференса. Из-за этого получается несколько разных предсказаний, откуда считается неопределенность.

## Источники
- Gal & Ghahramani, ICML 2016
- Hüllermeier & Waegeman, arXiv 2020
